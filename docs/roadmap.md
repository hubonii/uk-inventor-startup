# Product & Development Roadmap

## Phase 1: Validation & Architecture (Current)
- Complete legal research on UK Gig Economy classification.
- Finalize system design and database architecture.
- Build financial model and unit economics spreadsheet.

## Phase 2: Core MVP Development (Months 1-3)
- Build the matching algorithm (PostGIS).
- Integrate Onfido API for identity and Liveness checks.
- Build the basic React Native client with the Camera Sandbox (for taking secure photos without saving to the local camera roll).
- Implement the Double QR-Code cryptographic exchange.

## Phase 3: Closed Beta (Month 4)
- Launch within a single closed network (e.g., University Campus).
- Subsidize the first 500 deliveries to test the geofencing and anti-hoarding alerts.
- Gather feedback on the £3-£5 payout model vs batching behavior.

## Phase 4: Public Launch & B2B Integration (Month 6)
- Open to general public in Zone 1 London.
- Launch the Web Dashboard for local retailers to batch-upload delivery intent.
- Introduce the "Gold Tier" and high-value item insurance.
