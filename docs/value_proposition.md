# Value Proposition

## 🌟 For Sendors (Clients)
- **Sustainability:** Send packages completely carbon-neutral.
- **Speed & Flexibility:** Same-day delivery without the premium courier price tag.
- **Security:** Double QR-code chain of custody, tamper-evident bags, and live GPS tracking ensure high security and trust.
- **Insurance:** Built-in micro-insurance for every package.

## 🌟 For Couriers (Commuters)
- **Effortless Earnings:** Monetize existing daily commutes without going out of your way.
- **No Gamification or Pressure:** "Intent-based" matching means you only deliver where you are already going. You dictate the route, not the algorithm.
- **Safety First:** Strict prohibition on contraband, visual content inspection before packaging, and SOS integration protect the courier.

## 🌟 For the Environment (UK Focus)
- **Zero Additional Emissions:** By piggybacking on existing human movement (metro, walking, daily car commutes), we add zero new vehicles to the road.
